**Structure-Based Virtual Screening of BCR-ABL Tyrosine Kinase Inhibitors**

**Overview**
This project aims to identify potential inhibitors of the BCR-ABL tyrosine kinase associated with Chronic Myeloid Leukemia (CML) using computational drug discovery approaches.

**Objectives**
- Screen potential kinase inhibitors
- Perform molecular docking
- Analyze protein–ligand interactions
- Evaluate ADMET properties

**Tools Used**
- AutoDock Vina
- Discovery Studio Visualizer
- OpenBabel
- SwissADME

**Results**
A promising lead compound was identified based on docking score, binding interactions, and favorable ADMET properties.
